\section{Contenu des livraisons}

\subsection{Livraison des versions 1 à 4}
Le livrable est composé des documents suivants :
\begin{itemize}
\item Le document d'analyse amendé
\item Le document de conception amendé
\item Un manuel d'installation
\item Un manuel d'utilisation
\item Un tutorial d'utilisation
\item Le code
\item Les données utilisées\\
\end{itemize}
Les livraisons du jeu se font au format Zip, contenant les répertoires suivants :
\begin{itemize}
\item \begin{tt}src\end{tt} : contenant les fichiers sources python du projet ;
\item \begin{tt}data\end{tt} : contenant les données utilisées par le programme ;
\item \begin{tt}lib\end{tt} : contenant les librairies python utilisées par le jeu ;
\item \begin{tt}doc\end{tt} : contenant les documents et tutoriaux décrivant le programme.\\
\item \begin{tt}db\end{tt} : contenant la base de données du programme (stockage dynamique).\\
\end{itemize}
Les fichiers contenus dans le répertoire \begin{tt}doc\end{tt} sont aux formats latex, pdf et jpg.
