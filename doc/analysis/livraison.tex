organisation  des livraisons
\begin{itemize}
	\item un répertoire {\tt src} contenant les scripts de l'application
	\item un répertoire {\tt lib} contenant les librairies externes utilisées pour des fonctionnalités de l'application
	\item un répertoire {\tt data} contenant les différentes données (images, textes, \ldots)
	\item un répertoire {\tt doc} contenant  les différentes documentations afférentes au projet.
\end{itemize}
